## 🚀 How to Get Started

### To View the Application
1. The dev server is currently running at **http://localhost:3000**
2. Open your browser and navigate to that URL
3. Explore all the features!

### To Stop and Restart
```bash
# Stop: Press Ctrl+C in the terminal
# Restart using the batch file:
cd d:\report
start-dev.bat
```

### To Deploy
```bash
npm run build
npm start
```
